    - start-turn: when voiceState transitions from any non-listening to 'listening', call capture.start() and begin accumulating frames in a local ref array. The capture instance is created fresh each turn (not reused across turns).
    - frame accumulate: each onPcmFrame(buf: ArrayBuffer) pushes buf into the per-turn frames ref array.
    - stop-turn flush: when voiceState transitions to 'processing' (stopTurn on PTT release), call capture.stop(), then concatenate all accumulated Int16 frames in order into a single ArrayBuffer by copying each frame's bytes. If the total byte length is 0, call session.endTurn() WITHOUT calling voiceFeedAudio.
    - flush send: if byte length > 0, call the injected voiceFeedAudio(concatenatedBuffer). Regardless of success or failure, call session.endTurn() after to release the gate. The feedAudio call is wrapped in try/catch; endTurn() is called in the finally-equivalent path so session never hangs in 'processing'.
    - bargeIn / back-to-idle from non-processing: when voiceState becomes 'idle' while a capture is active (i.e. captureRef.current is non-null), call capture.stop() and DISCARD the buffer — no voiceFeedAudio call.
    - onError: call capture.stop(), discard buffer, dispatch a toast via the CustomEvent bus:
        window.dispatchEvent(new CustomEvent('aria:toast', { detail: { kind: 'error', message: err.message } }))

    The hook uses useRef for the capture handle (captureRef) and the per-turn frames array (framesRef). The session parameter is the flat VoiceSessionState & VoiceSessionActions snapshot passed in by the caller — no store import needed. Drive the lifecycle with a useEffect keyed on [session.voiceState]:

      useEffect(() => {
        const curr = session.voiceState;
        const prev = prevStateRef.current;
        prevStateRef.current = curr;

        if (prev !== 'listening' && curr === 'listening') {
          // Start a fresh turn
          framesRef.current = [];
          const handle = opts.createCapture({
            onPcmFrame(buf) { framesRef.current.push(buf); },
            onError(err) {
              void handle.stop();
              framesRef.current = [];
              if (typeof window !== 'undefined' && typeof window.dispatchEvent === 'function') {
                window.dispatchEvent(new CustomEvent('aria:toast', { detail: { kind: 'error', message: err.message } }));
              }
              session.endTurn();
            },
          });
          captureRef.current = handle;
          void handle.start();
        } else if (prev === 'listening' && curr === 'processing') {
          // Stop + flush
          const handle = captureRef.current;
          captureRef.current = null;
          const frames = framesRef.current;
          framesRef.current = [];
          void (async () => {
            if (handle) await handle.stop();
            const totalBytes = frames.reduce((n, f) => n + f.byteLength, 0);
            if (totalBytes === 0) { session.endTurn(); return; }
            const out = new Uint8Array(totalBytes);
            let offset = 0;
            for (const f of frames) { out.set(new Uint8Array(f), offset); offset += f.byteLength; }
            try { await opts.feedAudio(out.buffer); } finally { session.endTurn(); }
          })();
        } else if (curr === 'idle' && captureRef.current !== null) {
          // bargeIn path: stop + discard
          const handle = captureRef.current;
          captureRef.current = null;
          framesRef.current = [];
          void handle.stop();
        }
      }, [session.voiceState]);
