## Footguns (documented for future devs)

1. **Stripe Payment Links default to one-time mode** even when the underlying Price is recurring. Must explicitly choose subscription mode in the Payment Link UI; otherwise the webhook never fires `customer.subscription.created`.
2. **`session.customer_email` is `null`** for hosted Checkout sessions. Must fall back to `session.customer_details.email`. (Both license rows verified the fallback path works.)
3. **`wrangler 3.x d1 migrations apply` defaults to `--local`.** Must pass `--remote` to hit production D1 — otherwise migrations silently land in the local miniflare DB and the deployed Worker 500s on missing tables.
4. **PowerShell `curl` is an `Invoke-WebRequest` alias** that prompts on JSON bodies. Use `curl.exe` (real curl, ships on Win11) or `Invoke-RestMethod` for smoke tests.
