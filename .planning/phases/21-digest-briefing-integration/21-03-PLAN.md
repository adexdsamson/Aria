    FUNCTION runDigest(deps: WhatsAppDigestDeps): Promise<void> — the async unit of work:
      1. Compute today = new Date().toISOString().slice(0,10)
      2. Compute windowStart = datetime('now', '-3 days') equivalent as ISO string: new Date(Date.now() - WINDOW_DAYS * 86400_000).toISOString()
      3. Get tracked groups: SELECT jid, display_name FROM whatsapp_group WHERE tracked=1
      4. If no tracked groups, return early
      5. Create a p-queue with concurrency:1
      6. For each tracked group, enqueue an async task that:
         a. Fetches messages in window: WITH last_digest AS (SELECT MAX(m.sent_at) AS watermark FROM whatsapp_message m INNER JOIN whatsapp_group_digest d ON d.jid=m.jid WHERE d.jid=? AND d.summary_text IS NOT NULL AND d.date < ?) SELECT m.jid, m.body_text, m.sent_at, m.sender_jid FROM whatsapp_message m WHERE m.jid=? AND m.sent_at >= COALESCE((SELECT watermark FROM last_digest), ?) AND m.sent_at >= ? ORDER BY m.sent_at ASC LIMIT 150 — using ISO string windowStart and today as parameters
         b. If message count < MIN_ACTIVITY, skip (do NOT write a row — D-10 sub-threshold omit)
         c. Call localModelFactory() synchronously to get model
         d. Wrap await gen({model, system: DIGEST_SYSTEM_PROMPT, prompt: buildGroupPrompt(...), temperature: 0}) in try/catch
         e. On success: INSERT OR REPLACE INTO whatsapp_group_digest (jid, date, summary_text, generated_at, model_id) VALUES (?,?,?,?,?) with generated_at=Date.now() and model_id=DEFAULT_LOCAL_MODEL
         f. On catch: log warn with scope:'whatsapp-digest', then INSERT OR REPLACE with summary_text=null, generated_at=null, model_id=null
      7. Await queue.onIdle() to wait for all groups
      8. Catch any outer error and log it — runDigest never throws to caller
