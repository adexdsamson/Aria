## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Renderer → Main IPC (SCHEDULING_PROPOSE, SCHEDULING_CONFIRM_TARGET, SCHEDULING_OVERRIDE, APPROVALS_APPROVE) | Untrusted NL strings + approvalIds + override reasons cross |
| Main → LLM (frontier or local) | NL command body sent for intent extraction |
| Main → Google Calendar API | Read (freebusy + instances) and write (via 04-01 chokepoint only on approve) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-03-01 | Information Disclosure | NL command containing PII routed to frontier LLM | mitigate | Phase 3 redactor pre-pass in parseIntent; router decides local-only if redaction flags PII (Pattern G) |
| T-04-03-02 | Information Disclosure | Frontier LLM sees attendee emails or calendar contents | mitigate | parseIntent prompt contains ONLY the NL command text + current date; resolver runs locally against calendar_event cache — frontier never sees event payloads |
