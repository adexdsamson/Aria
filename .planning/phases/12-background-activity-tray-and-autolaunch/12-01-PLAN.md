| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01 | Tampering | BG_SET_PREFS payload — unexpected keys / non-boolean values | mitigate | Closed Zod schema in registerBackgroundHandlers; extra keys rejected, non-boolean rejected |
| T-12-02 | Elevation | Silent auto-launch enable | mitigate | All toggles are user-visible in BehaviourSection; no auto-enable on update; default `autoLaunch=false` (D-02) |
| T-12-03 | Tampering | setLoginItemSettings path/args injection | mitigate | `args` is the hardcoded string array `['--was-auto-launched']`; `path` not passed (defaults to `process.execPath`) |
| T-12-04 | Information Disclosure | Background prefs read pre-unlock returns wrong defaults | accept | Defaults are conservative: closeToTray=true (no data leak), autoLaunch=false, notificationsEnabled=true. DB wins on first unlock via reconciler. |
</threat_model>
