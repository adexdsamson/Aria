<constraints>
- Every non-refusal answer must include at least one citation.
- No-source behavior is a hard refusal: `I couldn't find anything in your data about that.`
- Answer synthesis must reuse Phase 3 sensitivity routing: general PII redacted for frontier use, HR/legal/financial severity >= medium answered locally.
- Citation clicks must open an inspectable source or preview with the cited span/snippet visible.
- Retrieval is unified across accounts by default, with an account filter if the existing UI pattern is cheap to reuse.
</constraints>
